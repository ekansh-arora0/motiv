# Motive AR — Parkinson's Adaptive Coach (Demo)

Static single-page demo showing how Motive AR glasses stream Parkinson's signals (tremor, gait, focus) into a caregiver-ready plan. Content is simulated for presentation and testing purposes.

## Run locally

Open `index.html` in a browser (double-click or serve with any static server). Everything is bundled as plain HTML/CSS/JS.

## What you see
- **Hero**: framing for Parkinson's adaptive AR therapy.
- **Signal snapshot**: simulated metrics from glasses (tremor, gait, focus, adherence).
- **Care plan**: daily actions plus caregiver checklist and medication schedule.
- **Session log & real-time feed**: example guided AR sessions and live signals.
- **Alerts**: proactive reminders for patient + caregiver.

## Customizing simulated data
Edit `index.html` (script section) to adjust telemetry parameters, event frequency, or simulated scenarios.
